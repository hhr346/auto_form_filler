# Auto Form Filler - 自动填充求职表单

Edge 浏览器插件，自动填充求职网站表单，告别重复填写！

## ✨ 功能特性

- ⚡ **一键填充** - 自动识别并填充表单字段
- 🎯 **智能匹配** - 基于关键词智能匹配表单字段
- 📝 **预设字段** - 姓名、身份证、手机、邮箱等常用字段
- ➕ **自定义字段** - 支持自由添加新字段
- 💾 **本地存储** - 数据安全存储在本地

## 📦 预设字段

## 🚀 安装方法

### 方法一：开发者模式安装

1. 打开 Edge 浏览器
2. 访问 `edge://extensions/`
3. 开启"开发者模式"
4. 点击"加载解压缩的扩展"
5. 选择 `auto-form-filler` 文件夹
6. 完成安装！

### 方法二：打包安装

```bash
# 在 Edge 扩展页面
1. 点击"打包扩展程序"
2. 选择项目根目录
3. 生成 .crx 文件
4. 拖拽到扩展页面安装
```

## 📖 使用指南

### 1. 配置个人信息

1. 点击浏览器工具栏的插件图标
2. 填写各项基本信息
3. 点击"保存数据"

### 2. 自动填充表单

1. 打开求职网站（如 BOSS 直聘、智联招聘等）
2. 进入需要填写的表单页面
3. 点击插件图标
4. 点击"一键填充"

### 3. 添加自定义字段

1. 点击"添加自定义字段"
2. 输入字段名称（如：期望薪资）
3. 选择字段类型（text/select）
4. 输入匹配关键词（用逗号分隔）
5. 如果是 select 类型，输入选项

## 🔧 开发说明

### 项目结构

```
auto-form-filler/
├── manifest.json          # 扩展配置文件
├── src/
│   ├── content.js        # 内容脚本（表单填充逻辑）
│   └── background.js     # 后台脚本（消息处理）
├── popup/
│   ├── index.html        # 弹出页面
│   └── popup.js          # 弹出页面逻辑
├── icons/                # 图标文件
└── README.md             # 说明文档
```

### 核心原理

1. **智能匹配**: 通过 label 文本、placeholder、name 属性等匹配字段
2. **事件触发**: 填充后触发 input/change 事件，确保表单验证通过
3. **本地存储**: 使用 chrome.storage.local 安全存储数据
